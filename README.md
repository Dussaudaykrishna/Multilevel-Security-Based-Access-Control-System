**Introduction**
          Security systems are essential for protecting sensitive areas such as lockers,
laboratories, and restricted rooms. Traditional lock systems are vulnerable because they rely only on a
physical key or single password authentication.
          To improve security, this project implements a Multilevel Security Based Access Control System
using LPC2138 ARM7 microcontroller. The system uses two levels of authentication:
           1) Password verification through a keypad.
           2) Password verification through a keypad.
**Objective**
  The main objectives of this project are:
        1) To design a secure access control system
        2) To implement two-factor authentication
        3) To interface keypad, LCD, UART, and motor driver with LPC2148.
        4) To generate and verify OTP for additional security
        5) To display system status using LCD display.**
**Hardware Components**
         LPC2138 Microcontroller
         4x4 Matrix Keypad
         16x2 LCD Display
         DC Motor
         LED
         Push Button Switch
         Virtual Terminal
         Power supply
